# 🔄 Restart Command

Instructions to restart Claude Code and clear all context.

## 📝 Manual Process

Since Claude cannot restart itself, follow these steps:

1. **Stop Claude**: Press `Ctrl + C` in terminal
2. **Restart**: Type `claude` and press Enter

## 🚀 Quick Restart Script

Run the restart helper:
```bash
.claude\scripts\restart_claude.bat
```

## 💡 Why Restart?

- Clear all conversation context
- Reset memory usage
- Fix stuck processes
- Start fresh with new task

## ⚡ Alternative: Clear Context

If you just want to clear conversation:
```bash
/clear
```
